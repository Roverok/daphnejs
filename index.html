Daphne.js (Github)

Library for creating, editing, and checking parse trees. Its only dependency is d3.

Daphne is currently in development mode and does not have a stable version.

Click a node and its new parent to move nodes within the parse tree.

Features

  • Ingests flat data with id and head attributes, and builds their parse tree.
  • Modes: Edit, Play (planned: display).
  • Zoom and pan trees.

Planned Features

Daphne.js will emulate and improve upon all the features in the OPP parse tree demo, including:

  • Answer-checking and game-play mode.
  • Export data to XML.
  • Change attributes on the nodes. Add and remove attributes as needed.
